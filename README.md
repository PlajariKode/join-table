# join-table
Menampilkan data dari 3 tabel yang berbeda dengan menggunakan PHP

# Import database
- buat database dengan nama **ex**
- klik pada menu import
- pilih file **ex.sql**

# Run
- jalankan `join-table.php` pada browser
